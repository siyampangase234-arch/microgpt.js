## üìù How to Use microgpt.js

Once the application is open in your browser, here is how to use it:

### Training Your GPT Model

1. **Prepare your text data:** Write or gather plain text you want the model to learn from. For beginners, try with simple sentences about a topic you like.

2. **Load the data:** Use the interface to paste or upload your text.

3. **Start training:** Click the ‚ÄúTrain‚Äù button. The model will process the text step-by-step.

4. **Training progress:** You will see progress updates in the browser. Training may take a few minutes based on your data size and computer speed.

### Generating Text (Inference)

After training:

1. Enter a starting word or phrase in the ‚ÄúPrompt‚Äù box.

2. Click ‚ÄúGenerate‚Äù to create new text. The model uses what it learned to predict and write more text.

3. Experiment with different prompts and see how the output changes.

## üîß Settings & Options

microgpt.js provides some basic settings for you to customize:

- **Training speed:** Choose how fast the model learns. Faster speed uses more CPU but finishes quicker.

- **Text length:** Set how many words you want to generate during inference.

- **Model size:** Adjust parameters to make the model simpler or more complex. Larger models need more memory.

- **Reset model:** Clear the trained model and start over with new data.

All settings can be changed from the user interface without coding.
